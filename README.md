## 🔥 The Problem

Nepal's agricultural supply chain is broken — and farmers pay the price.

- 🚫 **No price visibility** — middlemen hide real market rates from farmers
- 🤝 **Verbal-only contracts** — no accountability, no paper trail, no protection
- 💸 **Income hijacking** — farmers lose a significant share of earnings to manipulation
- 📉 **Zero transparency** — customers overpay while farmers are underpaid

> *The person who grows your food often earns the least from it.*

---

## ✅ Our Solution

**Hamro Krishi** is a three-sided mobile platform that puts every party on equal footing.

| Feature | What it does |
|---|---|
| 📊 **Real-time Price Feed** | All three parties see the same live market prices — no information asymmetry |
| 📝 **Digital Contracts** | Replace verbal agreements with recorded, tamper-proof digital contracts |
| 🔗 **Three-sided Marketplace** | Farmers, middlemen, and customers connected in one seamless platform |
| 🚫 **Anti-corruption by Design** | System architecture that eliminates room for price manipulation |

**Core belief: When everyone sees the same data, no one can cheat.**

## 🛠️ Tech Stack

### Mobile (Frontend)
- **Flutter / Dart** — cross-platform mobile app (iOS & Android)
- Clean architecture with separation of concerns
- Responsive UI for low-end Android devices common in rural Nepal

### Backend
- **Node.js** — REST API server
- Deployed on **Render**

## 🚀 Getting Started

### Prerequisites
- Flutter SDK `>=3.0.0`
- Dart `>=3.0.0`
- Node.js `>=18.x`
- npm or yarn

### Clone the Repository
```bash
git clone https://github.com/Amit-Bajracharya/Hamro_Krishi.git
cd Hamro_Krishi
```

### Run the Mobile App
```bash
cd mobile
flutter pub get
flutter run
```

### Run the Backend
```bash
cd backend
npm install
npm start
```
